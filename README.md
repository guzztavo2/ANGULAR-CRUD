# ANGULAR CRUD

### Requisitos para testar o projeto:
- 1 - Instalar [NodeJS](https://nodejs.org/en/);
- 2 - Instalar o [Angular](https://angular.io/);
- 3 - Instalar o [Angular Material](https://material.angular.io/);
